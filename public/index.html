DigitalOcean App Platform gRPC Demo

About


Consists of two services.

edge

The HTTP gateway service which takes HTTP requests and proxies to gRPC service.

grpcsvc

The actual gRPC service that handles the call.

Usage


http POST 'https://sample-node-grpc-edge-5h428.ondigitalocean.app/' name=foo

Proto

syntax = "proto3";

package helloworld;

service Greeter {
    rpc SayHello (HelloRequest) returns (HelloReply) {}
}

message HelloRequest {
    string name = 1;
}

message HelloReply {
    string message = 1;
}
